## Issue: Formatting Not Applied

If you see raw HTML or unformatted text:

1. **Check field type** — Custom.PrerequisiteforTest must be **HTML** in ADO Process (Organization Settings → Process → Fields).
2. **Where you view** — Open the work item in full form; grid/list views may not render HTML.
3. **Restart MCP** — After server updates, restart Cursor or reload MCP.
